Command for bundeling the application

Must be run with in the virtual environment from the root of the project (where the main.py file is located outside of the video_scoring folder)

python -m PyInstaller --noconfirm --log-level=WARN --windowed --name "Video Scoring" --add-data="video_scoring/resources/*;video_scoring/resources/" --hidden-import qtpy --icon="video_scoring\resources\icon_gray.ico" main.py
